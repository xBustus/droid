# Droid Simulator - Java

### Descrição do Projeto
Este projeto consiste numa simulação de robôs (Droids) desenvolvida em Java para fins de estudo de Programação Orientada a Objetos (POO). O sistema permite a criação de múltiplas instâncias de robôs, execução de tarefas que consomem energia e a interação direta entre objetos para transferência de bateria.

### Funcionalidades Implementadas
* **Gestão de Instâncias:** Criação de múltiplos robôs com identidades e estados de bateria independentes.
* **Sistema de Bateria:** Métodos que controlam o consumo de energia conforme as ações são executadas.
* **Interação entre Objetos:** Lógica de transferência de energia onde um objeto altera o estado de outro objeto recebido via parâmetro.
* **Relatório de Estado:** Exibição formatada de dados do sistema através da consola.

### Conceitos Técnicos Aplicados
* **Classes e Objetos:** Estruturação da planta do sistema e instanciação através da palavra-chave `new`.
* **Escopo e Atribuição:** Manipulação de variáveis de instância e parâmetros de construtor.
* **Métodos Void e String:** Diferenciação entre métodos de ação (void) e métodos de retorno de dados.
* **Sobrescrita de Métodos (Override):** Implementação do método `toString()` para representação textual personalizada do objeto.

### Como Executar
1. Certifique-se de que tem o JDK (Java Development Kit) instalado.
2. Clone o repositório ou descarregue o ficheiro `Droid.java`.
3. Compile o ficheiro: `javac Droid.java`
4. Execute o programa: `java Droid`
